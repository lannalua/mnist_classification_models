# Mnist Classification Models

Esse repositório mostra a aplicação dos meus estudos de Modelos de Classificação. O objeto de estudo é o dataset MNIST, clássico para introdução aos estudos de Machine Learnig e Deep Learning. 

## Modelos Utilizados
- Regressão Logística
- MLP - Multilayer perceptron
- CNN - Convolutional Neural Network

## Métodos de Otimização Utilizados

- GS: Grid Search
- RS: Random Search
- Bayesiana
- Hyperband
- GJO: Golden Jackal Optmization

### Divisão de Pastas

- \cnn - modelos treinados com CNN
- \mlp - modelos treinados com MLP
